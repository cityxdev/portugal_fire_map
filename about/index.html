Dados históricos de incêndios - Sobre

Base para análises comparativas, nas dimensões temporal e geográfica, das causas, áreas e combate, a partir dos dados de histórico do ICNF para os incêndios registados em Portugal Continental.

Objetivo

O objetivo deste website é contribuir com dados estatísticos para a discussão pública sobre os incêndios em Portugal Continental.
Desta forma, pode ser visto como um complemento ao pré-existente fogos.icnf.pt, utilizando este website a mesma fonte de dados, como explicado abaixo.
Não é propósito deste website fornecer dados em tempo real — esse tipo de informação já pode ser consultado aqui.

Fonte de dados

A principal fonte de dados deste website é o webservice do ICNF que alimenta também a página fogos.icnf.pt.
Esta fonte não é utilizada diretamente, mas antes os dados são processados e agregados numa base de dados relacional com suporte para informação geográfica. Este processo é open source e o código fonte pode ser consultado aqui.
